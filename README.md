# Valyria | Premium Digital Marketing & Industrial Solutions

Valyria is a modern, high-performance web agency landing page built with **React**, **TypeScript**, and **Tailwind CSS**. It showcases a diverse portfolio ranging from industrial energy solutions (CHAMS) to professional technical repairs (FYXTERS).

---

## 🚀 Features

* **Responsive Design:** Fully optimized for all screen sizes (Mobile, Tablet, Desktop).
* **Modern UI/UX:** Clean, minimalist, and luxury aesthetic using a black-and-gold theme.
* **Dynamic Sections:** * **Work Section:** A zig-zag layout featuring project details and key performance stats.
    * **Animations:** Smooth reveal effects powered by `framer-motion`.
* **SEO Optimized:** Meta tags and Open Graph configuration for better visibility.

---

## 🛠️ Tech Stack

* **Frontend:** React 18 + Vite
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Language:** TypeScript
* **Icons:** Lucide React (optional)
